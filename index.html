Fullstack Mobile Development Punggawa Store

Aplikasi Punggawa Store adalah aplikasi e-commerce yang dibangun dengan Flutter untuk memberikan pengalaman belanja online yang mudah dan nyaman. Aplikasi ini menggunakan Laravel untuk backend dengan MySQL sebagai databasenya, serta mengimplementasikan sistem autentikasi menggunakan Laravel Jetstream yang mendukung dua jenis pengguna, yaitu user biasa dan admin, serta menyediakan login dengan akun Google melalui Firebase. Pengguna dapat melakukan pembayaran dengan berbagai metode melalui Midtrans dan menghitung ongkos kirim menggunakan API Rajaongkir. Fitur aplikasi ini meliputi keranjang belanja untuk menambahkan produk, wishlist untuk menyimpan produk favorit, dan chat dengan admin untuk berkomunikasi terkait pertanyaan atau bantuan.
